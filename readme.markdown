#Unofficial Github Widget

A Github widget like 'Facebook Like Box'.

###Usage

Code:

```
<iframe src="github-widget.html?user=twitter&repo=bootstrap" allowtransparency="true" frameborder="0" scrolling="0" width="292" height="290"></iframe>
```
